# BeyondAbilityInstitution
七日殺專用的模組；超能機構

BEABIN 為主模組, 一定要安裝.

BEABIN_BedRecoveryHealth 為附加模組,無強制使用.
用途: 當玩家站在床上時,會逐漸回復生命力與耐力.
  
BEABIN_DropOtherMinerals 為附加模組,無強制使用.
用途: 挖掘礦脈時,有機率性多掉出銅礦.
  
BEABIN_HighestLevel 為附加模組,無強制使用.
用途: 提升玩家等級上限至999級.
  
BEABIN_PickUpBlocks 為附加模組,無強制使用.
用途: 製訂某些方塊可以被玩家撿起來帶回家.

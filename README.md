<h1 id="beyond-abilityi-nstitution">Beyond AbilityI nstitution</h1>
<p>七日殺專用的模組：<strong>超能機構</strong><br>
master代表最新的版本。</p>
<p><strong>目前最新版本</strong>：18.4</p>
<ul>
<li>
<p>BEABIN<br>
為主模組，一定要安裝。</p>
</li>
<li>
<p>BedRecoveryHealth<br>
為附加模組，非強制安裝。<br>
用途：當玩家站在床上時，會逐漸回復生命力與耐力。</p>
</li>
<li>
<p>DropOtherMinerals<br>
為附加模組，非強制安裝。<br>
用途：挖掘礦脈時，有機率性多掉出銅礦。</p>
</li>
<li>
<p>HighestLevel<br>
為附加模組，非強制安裝。<br>
用途：提升玩家等級上限至999級。</p>
</li>
<li>
<p>PickUpBlocks<br>
為附加模組，非強制安裝。<br>
用途：製訂某些方塊可以被玩家撿起來帶回家。</p>
</li>
</ul>
<p><strong>下載方法</strong>：先選擇左上角的版本（例如：master），再點擊右上角的「Code」綠色按鈕，再選擇「Download ZIP」就會立即下載了，下載回去的檔案名稱為BeyondAbilityInstitution-XXX.zip，再將你需要的模組目錄（例如：BEABIN）整個解壓縮到你的MOD目錄即可。</p>

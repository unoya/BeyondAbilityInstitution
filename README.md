<h1 id="beyond-abilityi-nstitution">Beyond AbilityI nstitution</h1>
<p>七日殺專用的模組；超能機構</p>
<ul>
<li>
<p>BEABIN 為主模組, 一定要安裝.</p>
</li>
<li>
<p>BedRecoveryHealth 為附加模組,無強制使用.<br>
用途: 當玩家站在床上時,會逐漸回復生命力與耐力.</p>
</li>
<li>
<p>DropOtherMinerals 為附加模組,無強制使用.<br>
用途: 挖掘礦脈時,有機率性多掉出銅礦.</p>
</li>
<li>
<p>HighestLevel 為附加模組,無強制使用.<br>
用途: 提升玩家等級上限至999級.</p>
</li>
<li>
<p>PickUpBlocks 為附加模組,無強制使用.<br>
用途: 製訂某些方塊可以被玩家撿起來帶回家.</p>
</li>
</ul>
